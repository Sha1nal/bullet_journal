What is a Note?
    A note is an object with these properties:
        note id
        note text
        status:
            open
            closed
        priority: rating of (1-5)
        date created
        expiry date
        type:
            task
            event
            note

What is a journal?
    A journal is the 'book' into which notes are entered. This book will consist of:
        A future log table
